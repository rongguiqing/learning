# 1.位运算
    - 位运算就是在二进制基础上进行逻辑运算
        - &：按位与运算
            - 变量 = 值1 & 值2
        - |：按位或运算
            - 变量 = 值1 | 值2
        - ~：按位或运算
            - 变量 = ~值
        - ^：按位抑或运算
            - 变量 = 值1 ^ 值2
        - <<：左移运算
            - 变量 = 值 << 左移位数
        - >>：右移运算
            - 变量 = 值 >> 右移位数
